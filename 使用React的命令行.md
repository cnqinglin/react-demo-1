# 1. 在一处创建一个目录，cd 这个目录
    比如： cd demo-react-redux-1
# 2.安装React
   ```
    npm i -g create-react-app
   ```
# 3. 生成React文件系统
    ```
        //最后一个点不能忘
        create-react-app .
    ```
# 4. 打开React
    ```
         npm start
    ```
* 到这React的使用环境已经全部结束
 
# 5. 如果再需要使用Redux的话
### 1. 安装（需要安装的话先中断 React ）
    ``` 
        npm install redux
    ```
### 2. 打开React
    ```
        //这是重新打开React的搭建环境
        npm start
    ```
